# Reversi Web (Flask + Vanilla JS)

Browser-based Reversi (Othello) using the original Python game logic for rules and AI.

## Run

1. Install dependencies:

```bash
pip install -r requirements.txt
```

2. Start the server:

```bash
python server.py
```

3. Open:

`http://127.0.0.1:5000/`

## Features

- Player vs Player (local on one board)
- Player vs AI (`easy`, `medium`, `hard`)
- Automatic pass handling
- Score and turn display
- Same Python move validation / flipping / minimax logic as the terminal version
